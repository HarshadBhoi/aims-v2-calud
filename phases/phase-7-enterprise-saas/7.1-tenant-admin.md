# 7.1 — Multi-Tenant Administration

**Parent**: [Phase 7 — Enterprise & SaaS](OVERVIEW.md)
**Estimated effort**: 3 days

---

## Tasks

- [ ] **Platform Admin Portal**: Superadmin dashboard (separate from tenant admin)
  - Tenant list with status, plan, user count, storage usage
  - Create/suspend/delete tenants
  - Impersonate tenant admin (for support)
  - Platform-wide metrics (total users, engagements, storage)
- [ ] **Tenant Admin Settings**:
  - Organization profile (name, logo, primary color)
  - User management (invite, deactivate, change roles)
  - Standard pack activation/deactivation
  - Feature flags (enable/disable modules)
  - Security settings (MFA enforcement, session timeout, password policy)
  - Data retention policies
- [ ] **Usage Dashboard**: Storage used, API calls, user seats, engagement count
- [ ] **Tenant Provisioning Automation**: New signup → tenant created → default config → welcome email
