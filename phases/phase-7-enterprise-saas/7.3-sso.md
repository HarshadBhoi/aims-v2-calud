# 7.3 — SSO & Enterprise Identity

**Parent**: [Phase 7 — Enterprise & SaaS](OVERVIEW.md)
**Estimated effort**: 3 days

---

## Tasks

- [ ] **SAML 2.0 Support**: Enterprise SSO via SAML (Azure AD, Okta, OneLogin, ADFS)
- [ ] **OIDC Support**: OpenID Connect for modern identity providers
- [ ] **JIT Provisioning**: Auto-create user account on first SSO login
- [ ] **Domain Verification**: Verify org domain ownership before enabling SSO
- [ ] **SSO Configuration UI**: Tenant admin can configure SAML/OIDC endpoints, certificates
- [ ] **Mixed Auth**: Support both SSO and email/password users in same tenant
- [ ] **SCIM Provisioning**: Auto-sync users from identity provider (create, update, deactivate)
- [ ] **Directory Sync**: Sync roles/groups from AD/Okta to AIMS roles
