# 7.5 — CI/CD & DevOps Pipeline

**Parent**: [Phase 7 — Enterprise & SaaS](OVERVIEW.md)
**Estimated effort**: 2-3 days

---

## Tasks

- [ ] **GitHub Actions CI**: Lint → type-check → unit tests → integration tests → build (on every PR)
- [ ] **Staging Deploy**: Auto-deploy `develop` branch to staging environment
- [ ] **Production Deploy**: Manual approval → deploy `main` to production (zero-downtime)
- [ ] **Database Migrations**: Auto-run Prisma migrations on deploy
- [ ] **Blue-Green Deployment**: Zero-downtime deployments with instant rollback
- [ ] **Environment Management**: dev, staging, production with isolated databases
- [ ] **Feature Flags**: LaunchDarkly or custom flags for gradual rollouts
- [ ] **Release Management**: Semantic versioning, changelog generation, release notes
- [ ] **Docker Registry**: Build and push Docker images to registry
- [ ] **Infrastructure as Code**: Terraform or Pulumi for cloud infrastructure
- [ ] **Load Testing in CI**: k6 performance tests on staging before production deploy
