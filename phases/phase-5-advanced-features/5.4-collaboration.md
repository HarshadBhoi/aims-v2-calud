# 5.4 — Real-Time Collaboration

**Parent**: [Phase 5 — Advanced Features](OVERVIEW.md)
**Estimated effort**: 3-4 days

---

## Tasks

- [ ] **Presence Indicators**: See who's currently viewing/editing the same entity
- [ ] **Live Updates**: Changes by one user immediately reflected for others (WebSocket/SSE)
- [ ] **Collaborative Rich Text**: Multiple users editing the same finding/report section (TipTap Collaboration + Yjs)
- [ ] **Comments & Mentions**: Comment on any entity, @mention users to notify them
- [ ] **Activity Feed**: Per-engagement activity stream (who did what, when)
- [ ] **Review Notes**: Inline review annotations on findings and reports
- [ ] **Lock/Unlock**: Prevent concurrent edits on critical documents (optional pessimistic locking)
- [ ] **Change Indicators**: Visual diff showing what changed since last view
