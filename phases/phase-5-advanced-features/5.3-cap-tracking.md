# 5.3 — Corrective Action Plan (CAP) Tracking

**Parent**: [Phase 5 — Advanced Features](OVERVIEW.md)
**Estimated effort**: 2-3 days

---

## Tasks

- [ ] **CAP Dashboard**: All CAPs across engagements with overdue highlighting
- [ ] **Status Pipeline**: Visual pipeline (Not Started → In Progress → Completed → Verified)
- [ ] **Evidence Upload**: Auditees upload completion evidence directly
- [ ] **Auditor Verification**: Auditor reviews evidence, marks as verified or rejected
- [ ] **Overdue Alerts**: Automated notifications when CAPs pass due date
- [ ] **Escalation**: Auto-escalate to supervisor after N days overdue
- [ ] **Follow-Up Engagement Link**: Create follow-up engagement from overdue CAPs
- [ ] **Trend Analysis**: CAP completion rates over time, by department, by finding type
- [ ] **Aging Report**: CAPs by age (0-30, 31-60, 61-90, 90+ days)
- [ ] **Auditee Portal**: Limited view for auditees to see their CAPs and submit updates
