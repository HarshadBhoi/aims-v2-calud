## Tasks

- [ ] **In-App Notifications**: Bell icon with unread count, notification dropdown, mark as read
- [ ] **Email Notifications**: Transactional emails via Resend/Nodemailer
- [ ] **Notification Types**:
  - Approval request received
  - Approval granted/rejected
  - Assigned to engagement
  - CAP approaching due date
  - CAP overdue
  - Comment/mention
  - Report published
  - Independence declaration required
  - CPE hours reminder
- [ ] **Notification Preferences**: Per-user settings (in-app, email, both, none per type)
- [ ] **Digest Option**: Daily/weekly summary email instead of individual notifications
- [ ] **Email Templates**: Branded HTML emails with action buttons
- [ ] **Webhook Integration**: Send notifications to Slack, Teams, or custom endpoints (future)
