# 5.2 — Staff, Time & CPE Tracking

**Parent**: [Phase 5 — Advanced Features](OVERVIEW.md)
**Estimated effort**: 3-4 days

---

## Tasks

### Staff Management
- [ ] Staff directory with profiles (title, certifications, skills, department)
- [ ] Skill matrix (which standards/tools each auditor is proficient in)
- [ ] Capacity planning (hours available vs allocated)
- [ ] Certification tracking (CPA, CIA, CISA, CFE with expiration dates)

### Time Tracking
- [ ] Daily timesheet entry (engagement, category, hours, description)
- [ ] Categories: Planning, Fieldwork, Reporting, Admin, Training, Leave
- [ ] Weekly timesheet view (Mon-Fri grid)
- [ ] Time approval workflow (submit → supervisor approve)
- [ ] Budget vs actual hours per engagement
- [ ] Overtime/utilization reports
- [ ] Export to payroll/billing systems (CSV)

### CPE Tracking (Continuing Professional Education)
- [ ] CPE record entry (date, topic, provider, hours, type)
- [ ] GAGAS requirements: 80hrs/2yr, 24hrs govt audit, 20hrs/yr minimum
- [ ] IIA requirements: 40hrs/yr for CIA certification
- [ ] Compliance dashboard with progress bars
- [ ] Alerts when approaching deadline with insufficient hours
- [ ] CPE certificate upload (evidence)
- [ ] Annual CPE summary report
