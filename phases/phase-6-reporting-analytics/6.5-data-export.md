# 6.5 — Data Export & Integration API

**Parent**: [Phase 6 — Reporting & Analytics](OVERVIEW.md)
**Estimated effort**: 2-3 days

---

## Tasks

- [ ] **CSV/Excel Export**: Export any list view to CSV or XLSX (engagements, findings, time, etc.)
- [ ] **Bulk Export**: Export all data for an engagement as a ZIP (metadata + workpapers + reports)
- [ ] **Public REST API**: Documented API for third-party integrations
  - API key authentication
  - Rate limiting
  - OpenAPI/Swagger documentation
  - Endpoints for: engagements, findings, recommendations, time entries
- [ ] **Webhook Outbound**: Push events to external systems
  - Finding created/updated
  - Approval status changed
  - Report published
  - CAP overdue
- [ ] **Import API**: Bulk import from CSV/Excel (engagements, findings, audit universe)
- [ ] **Power BI / Tableau Connector**: ODBC or API-based data source for BI tools (future)
