# 3.3 — GAGAS Independence & Ethics

**Parent**: [Phase 3 — GAGAS Standard Pack](OVERVIEW.md)
**Estimated effort**: 2 days

---

## Objective

Implement GAGAS independence and ethics requirements (Chapter 3). Auditors must declare independence before starting fieldwork.

---

## Tasks

### 3.3.1 — Independence Declaration Form (§3.26)
- [ ] Port from AIMS v1 (`IndependenceDeclaration.tsx`):
  - Personal independence confirmation
  - Organizational independence confirmation
  - Financial interest disclosure
  - Non-audit service disclosure
  - Prior employment with auditee disclosure
  - Family relationship disclosure
  - External impairment disclosure
- [ ] E-signature (name, date, role)
- [ ] Required for all engagement team members before fieldwork
- [ ] Stored per user per engagement

### 3.3.2 — Independence Monitoring
- [ ] Dashboard widget: independence declarations status per engagement
- [ ] Alert when team member hasn't completed declaration
- [ ] Annual independence reminder (for ongoing engagements)
- [ ] Conflict of interest flagging

### 3.3.3 — Ethics & Professional Standards
- [ ] CPE requirement tracking (§4.16):
  - 80 hours per 2-year cycle
  - 24 hours in government auditing
  - 20 hours per year minimum
- [ ] Competency requirements per engagement type (§4.02)
- [ ] Ethics acknowledgment form

- Independence is GAGAS-specific but the form framework is reusable
