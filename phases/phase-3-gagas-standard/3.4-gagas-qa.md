# 3.4 — GAGAS QA & Peer Review

**Parent**: [Phase 3 — GAGAS Standard Pack](OVERVIEW.md)
**Estimated effort**: 2-3 days

---

## Objective

Implement GAGAS quality assurance (§5.01) and peer review requirements — the 60+ item QA checklist and peer review framework from AIMS v1.

---

## Tasks

### 3.4.1 — QA Checklist (§5.01)
- [ ] Port 60+ item checklist from AIMS v1 (`qaChecklistConfig.ts`) organized by category:
  - General Standards (independence, competence, quality control)
  - Planning (objectives, scope, methodology, risk assessment)
  - Fieldwork (evidence, analysis, documentation)
  - Reporting (content, clarity, accuracy)
  - Follow-Up (prior findings, corrective actions)
- [ ] Each item: text, GAGAS reference (§X.XX), required/optional, response type (Yes/No/NA)
- [ ] Checklist completion tracking (% complete, items needing attention)
- [ ] QA reviewer assignment per engagement

### 3.4.2 — QA Review Interface
- [ ] Checklist form: grouped by category, collapsible sections
- [ ] For each item: response (Yes/No/NA), comments, linked workpaper reference
- [ ] Overall QA rating: Satisfactory / Needs Improvement / Unsatisfactory
- [ ] Summary of issues found
- [ ] QA reviewer sign-off (name, date, role)

### 3.4.3 — Peer Review Module
- [ ] External peer review tracking:
  - Peer review organization
  - Review type (System, Engagement, Desk)
  - Review cycle dates
  - Rating received
  - Findings and recommendations
  - Corrective action status
- [ ] Internal peer review between engagement teams:
  - Reviewer assignment
  - Review checklist (subset of QA checklist)
  - Comments and suggestions
  - Resolution tracking

### 3.4.4 — QA Dashboard
- [ ] QA status across all engagements
- [ ] Overdue QA reviews
- [ ] Common QA findings (trend analysis)
- [ ] Peer review cycle status

---

## Notes

- QA is mandatory under GAGAS §5.01 — the system must enforce it
- Port the exact 60+ items from AIMS v1 qaChecklistConfig.ts
- Other standards have similar QA requirements (IIA Standard 1300) — reuse the checklist engine
