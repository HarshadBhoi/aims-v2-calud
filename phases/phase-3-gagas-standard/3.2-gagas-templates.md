# 3.2 — GAGAS Templates & Checklists

**Parent**: [Phase 3 — GAGAS Standard Pack](OVERVIEW.md)
**Estimated effort**: 3-4 days

---

## Objective

Create all GAGAS-specific templates (planning memo, work programs, report structure) and checklists that are loaded by the core engine when GAGAS is selected.

---

## Tasks

### 3.2.1 — Audit Planning Memorandum Template (§7.05-7.10)
- [ ] 14-section structured template (port from AIMS v1):
  1. Engagement Authorization
  2. Background and Context
  3. Audit Objectives
  4. Scope and Limitations
  5. Methodology and Approach
  6. Criteria and Standards
  7. Risk Assessment
  8. Internal Controls Assessment
  9. Materiality Considerations
  10. Staff Requirements and Competencies (§4.02)
  11. Timeline and Milestones
  12. Budget and Resource Allocation
  13. Communication Plan
  14. Supervision and Review Plan
- [ ] Each section: label, description, helpText, isRequired, defaultContent (starter text)
- [ ] Section references to GAGAS paragraphs

### 3.2.2 — Work Program Templates
- [ ] Financial Audit work program template:
  - Revenue cycle testing
  - Expenditure cycle testing
  - Payroll and personnel testing
  - Cash and investments testing
  - Internal controls evaluation
- [ ] Performance Audit work program template:
  - Program effectiveness evaluation
  - Efficiency and economy analysis
  - Compliance testing
  - Data analytics procedures
- [ ] Each template: ordered procedure steps with objective, approach, expected evidence

### 3.2.3 — Report Structure Template (§6.02)
- [ ] Required report elements:
  1. Report Title
  2. Objectives, Scope, and Methodology
  3. Background
  4. Audit Results (Findings)
  5. Conclusions
  6. Recommendations
  7. Agency Comments (Management Response)
  8. Auditor's Analysis of Agency Comments
  9. Confidential Information Handling
  10. Report Distribution
- [ ] Optional sections: Executive Summary, Appendices, Compliance Statement, Glossary
- [ ] Each section tagged as required/optional per engagement type

### 3.2.4 — Summary Schedule Template (§6.50)
- [ ] Findings summary table: finding number, title, criteria, condition, recommendation, status
- [ ] Auto-populated from engagement findings
- [ ] PDF format matching government report standards

### 3.2.5 — Supervisory Review Checklist
- [ ] Planning phase review items
- [ ] Fieldwork review items
- [ ] Reporting review items
- [ ] Evidence sufficiency review
- [ ] All items linked to GAGAS paragraphs

---

## Notes

- Templates are TypeScript objects (not files) — they're code that the engine interprets
- Templates can be customized per tenant (override sections, add custom sections)
- Port the 14-section APM from AIMS v1 (components/engagements/tabs/ patterns)
- Work program templates should be cloneable and editable — they're starting points, not rigid forms
